## ğŸš€ Overview

**Elegant ChatGPT Navigation** adds a persistent, auto-updating navigation panel to the ChatGPT interface.
It analyzes the structure of a conversation and builds a clean tree containing:

* **Prompts**
* **Responses**
* **H1 headings**
* **Code blocks** (with language + first two lines)

Scrolling through the chat automatically updates the navigator.
Clicking an item scrolls the chat to exactly the right position.

Perfect for developers, researchers, and heavy ChatGPT users.
